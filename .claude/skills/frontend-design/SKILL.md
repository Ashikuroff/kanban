# Frontend Design Skill

This skill provides assistance with frontend development tasks including component creation, styling, state management, and UI/UX improvements.

## Capabilities

- Component creation and modification
- Styling with CSS/Tailwind/SCSS
- State management patterns
- UI/UX improvements
- Responsive design implementation
- Accessibility enhancements

## Usage

Invoke this skill when working on frontend tasks:
```
/frontend-design [task description]
```

Examples:
- `/frontend-design Create a reusable button component with variants`
- `/frontend-design Improve the accessibility of the navigation menu`
- `/frontend-design Implement dark mode toggle functionality`
- `/frontend-design Optimize component re-renders for better performance`

## Guidelines

When using this skill:
1. Follow existing codebase patterns and conventions
2. Maintain consistency with the current styling approach (Tailwind CSS)
3. Ensure components are reusable and composable
4. Consider accessibility (WCAG) guidelines
5. Test responsive behavior across different screen sizes
6. Keep state management predictable and testable

## Best Practices

- Create small, focused components that do one thing well
- Use composition over inheritance for component reuse
- Leverage React hooks for state and side effects
- Implement proper TypeScript typings for props and state
- Write components that are easy to test in isolation
- Follow the existing file organization structure
